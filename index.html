    MyCafé

    PHP CodeIgniter Bootstrap MariaDB

    Website abrangente para cafeteria, seguindo a arquitetura MVC, e empregando. A plataforma oferece uma experiência completa, incluindo página inicial e loja, carrinho de compras, gestão de produtos e categorias, e um sistema de pedidos e entregas eficiente.     AgendaMD

    Python Kivy NumPy PyBabel

    Este projeto é um aplicativo de agenda nativo para Windows. A aplicação incorpora Material Design, tema escuro, e permite a criação e gerenciamento de tarefas e eventos com visualizações em calendário e lista. Conta com suporte a múltiplas linguagens via Multilingual internationalization com alteração dinâmica.     Hello API

    PHP MySQL Debian

    Este projeto envolve a criação de uma API de aplicativo de mensagens instantâneas. Uma experiência crucial foi a implantação e operação em um servidor Debian na indigoOcean, proporcionando conhecimentos práticos em gerenciamento de servidores, configurações de ambiente de produção e troubleshooting.

    ResumeCV

    Uma plataforma web para criação de sites portfólio e currículos online altamente personalizáveis. Sua página pública e landing page usam Blade (MVC).
